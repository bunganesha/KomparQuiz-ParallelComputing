# Task Parallelism - Python

Program ini mengimplementasikan task parallelism menggunakan thread di Python.

## Konsep:
- Fork → pembuatan thread
- Parallel Execution → task berjalan bersamaan
- Join → menunggu semua thread selesai

## Task:
1. Menghitung rata-rata
2. Menjumlahkan angka ganjil
3. Mencari kelipatan 3

## Cara Menjalankan:
python taskParallelism.py
